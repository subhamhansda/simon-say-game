# simon-say-game
It's a game built to increase memory made from HTML,CSS and Javascript.

There are four colourful boxes in the game.
When you start the game a random colour will flash then if you presses the same colour you will been landing into the next level again the same thing, a random colour will flash ,here you have to press two boxes the box flashed in the first level and the box in the 2nd level following the same sequence.The correct sequence will made you reach to the next level or otherwise you will be ending the game with getting the current level as your score.
